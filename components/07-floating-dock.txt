Component: Floating Dock (macOS-style)
Forrás stílus: Aceternity UI / 21st.dev
Mire jó: Diszkrét, lebegő navigáció az oldal alján (vagy oldalt mobilra). Ikonok hover-re felnagyulnak, mint a macOS Dock. Nem vonja el a figyelmet a fotókról.

Függőségek:
- React
- Tailwind CSS
- framer-motion
- lucide-react (ikonok: Home, Image, User, Mail, Instagram)

PROMPT:

Build a Floating Dock component (Mac OS Dock style).
- Two variants exported: FloatingDockDesktop and FloatingDockMobile, plus a wrapper FloatingDock that picks based on screen size (hidden md:flex / flex md:hidden).
- Desktop:
  - Horizontal pill: rounded-2xl, bg-gray-50 dark:bg-neutral-900, backdrop-blur, mx-auto, fixed bottom-6 left-1/2 -translate-x-1/2, z-50.
  - Track mouseX with framer-motion's useMotionValue across the dock.
  - For each icon, calculate distance from mouseX to that icon's center; map distance via useTransform to width/height (40px far, 80px close — useSpring smoothing).
  - Show a tooltip above the hovered icon (small dark pill with the link label, slides up + fades in).
- Mobile:
  - Single round button bottom-right with menu icon.
  - On click expands to a vertical column of icon buttons with stagger animation.
- Each item: `{ title, icon, href }`. Wrap in <Link>.

File: components/ui/floating-dock.tsx

Usage:
<FloatingDock items={[
  { title: "Főoldal", icon: <Home />, href: "/" },
  { title: "Portfolio", icon: <Image />, href: "/portfolio" },
  { title: "Rólam", icon: <User />, href: "/about" },
  { title: "Kapcsolat", icon: <Mail />, href: "/contact" },
  { title: "Instagram", icon: <Instagram />, href: "https://instagram.com/..." },
]} />
